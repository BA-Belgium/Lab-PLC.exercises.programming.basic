# Collection of Exercises for PLC Programming

## About this repository
This repository contains UNOFFICIAL UNSUPPORTED sample code for TwinCAT developed by [Beckhoff Automation](https://www.beckhoff.com). The sample codes in the repository are proposed simple solutions for the exercices of the book "Collection of Exercises for PLC Programming" by Tom Mejer Antonson.
The sample code is provided as-is under the Zero-Clause BSD license.
This sample code does not replace any official documentation or training. Please refer to the local Beckhoff subsidiary for the complete training session before even considering yourself a TwinCAT specialist.

## Technologies/Versions 
Technologies used within the project:

* [TwinCAT](https://www.beckhoff.com/nl-be/products/automation/twincat/texxxx-twincat-3-engineering/te1000.html): Version >=3.1.4026

## Usage
   The exercise were made with Beckhoff TwinCAT in mind and were tested in TwinCAT 4026.19. All code can be run in the TC1700 User Mode runtime.  

## Disclaimer
   In no way does this repository claim to contain the perfect solutions for the exercise.
   Samples re-use the code made in previous exercises. The re-usable code is found under folder _Library.
   The code is kept as simple as possible so it can be understood by any starter. 
   It has preparations for implementing OOP concepts but does not use any.
